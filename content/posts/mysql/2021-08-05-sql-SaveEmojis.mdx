# 배경
카카오 싱크를 사용하면서...
카카오톡 사용자 닉네임에는 생각보다 이모티콘이 많았다.
이모지(Emoji)를 DB에 저장해야 했다.

# 해결법
 mysql의 Charset 중 utf8과 utf8mb4 있다
mysql table 의 `Columns` 속성중 `Charset` 에
`utf8mb4` 을 설정하면 이모티콘을 추가할 수 있다.
