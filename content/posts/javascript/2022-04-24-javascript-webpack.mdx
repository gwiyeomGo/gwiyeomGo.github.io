# 배경

프론트엔드에 필수적으로 웹펙 개념을 많이 물어보는 거 같다.
개념을 알아보자~

# webpack 을 알기전에..
> Webpack is a free and open-source module bundler for JavaScript.
> module bundler?

1.모듈(module)
자바스크립트 파일을 기능 단위로 모듈화하고
이것을 하나로 관리할 방법이 필요해졌다.

2. 번들러(bundler)
의존성이 있는 모듈들을 하나(또는 여러 개)의 파일로 만들어주는 도구


# webpack 이 뭔가요?
webpack 은 번들러다.
