# sort.Slice
* 오름차순 (ascending order)
```
a := []int{5, 3, 4, 7, 8, 9}
sort.Slice(a, func(i, j int) bool {
    return a[i] > a[j]
})
for _, v := range a {
    fmt.Println(v)
}
```
* 내림차순
```
a := []int{5, 3, 4, 7, 8, 9}
sort.Slice(a, func(i, j int) bool {
    return a[i] < a[j]
})
for _, v := range a {
    fmt.Println(v)
}
```

# sort.Reverse
```
chars := []string{"a", "b", "c"}
sort.Sort(sort.Reverse(sort.StringSlice(chars)))
//[c b a]
```
