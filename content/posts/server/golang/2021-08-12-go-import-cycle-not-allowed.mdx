### 배경
1. serviceA 에서 다른 serviceB 의 함수를 호출하려고 한다.
2. serviceB에서 이미 serviceA를 호출하고 있다.
3. 코드가 실행되지 않는다.

결국 serviceA에 함수를 serviceB에 옮기고 사용했다

이럴경우 어떻게 해야 할까?


### golang 에서 순환참조 발생하는 경우

```
__________       __________
|         |<---- |        |
|packageA |      |packageB|
|         | ---->|        |
__________       __________
```

```
     packageA
    ↗        ↘
packageC ← packageB
```
