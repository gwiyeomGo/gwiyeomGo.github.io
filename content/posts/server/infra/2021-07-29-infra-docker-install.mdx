# 실행중인 이미지 중지

`docker ps` 로 상태 확인하고 중지`docker stop {containerID}` ,삭제`docker rm {containerID}` 했다
이렇게하니 제대로 종료되어서 재설치시 포트 중복 문제는 없어졌다
