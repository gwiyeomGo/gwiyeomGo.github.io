### time package

> time.Now()

- `t := time.Now()` 현재 시간을 알 수 있다

> time.Now().Format("20060102150405")

-  `t := time.Now()`형태를 yyyyMMddHHmmss 로 변경하고 싶다면` `t.Format("20060102150405")`

> year := time.Now().Year()

> endDate := time.Date(year, time.December, 31, 0, 0, 0, 0, time.UTC).Format("2006-01-02")

> startDate := time.Date(year, time.January, 1, 0, 0, 0, 0, time.UTC).Format("2006-01-02")

> currentYear, currentMonth, _ := time.Now().Date()
result.ThisYear = int64(currentYear)
result.ThisMonth = int64(currentMonth)
if currentMonth == 12 {
    result.NextMonth = int64(1)
}else {
    result.NextMonth = int64(currentMonth + 1)
}
