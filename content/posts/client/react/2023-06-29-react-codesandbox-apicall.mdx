# 배경
npm 컴포넌트 추가하고 demo 코드를 바로 볼 수 있도록
코듣센드 박스에 자동으로 업로드 되는 코드를 추가했다

# 코듣센드 박스에  업로드 방법

1. 설치
` npm install -g codesandbox `

2. 아래 코드를 실행하면 코드센드박스에 프로젝트가 업로드된다.
`codesandbox ./`

```
fetch("https://codesandbox.io/api/v1/sandboxes/define?json=1", {
    method: "POST",
    headers: {
        "Content-Type": "application/json",
        Accept: "application/json"
    },
    body: JSON.stringify({
        files,
        'package.json': {
            content: { dependencies: {} },
        }
    })
})
    .then(x => x.json())
    .then(({ sandbox_id }) => {
        const sandboxURL = `https://codesandbox.io/s/${sandbox_id}`;
         });
```
