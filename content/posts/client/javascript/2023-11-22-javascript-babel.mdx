전용 변환기 사용합니다.

* es6 이상의 최신 문법을 쓰자니 작성한 코드가 일부 실행 환경에서 동작하지 않음
* 자바스크립트 트랜스파일러 바벨을 사용해서 아직 최신 문법들이 적용되지 않은 실행 환경에서도 문제없이 작동시킨다

* TypeScript,JSX(React는 일반적으로 라는 특수한 문법) 로 작성된 코드 변환시 사용
