# 배경
moment 를 주로 사용했는데 더이상 업데이트 하지 않는 라이브러리를 사용하는 것은 아닌거 같아서
새로운 라이브러리를 찾아보니 dayjs 를 많이 사용한다는 것을 알게 되었다.


#  [dayjs](https://www.npmjs.com/package/dayjs)
JS 날짜 관련 라이브러리로 `npm i dayjs` 로 설치

### 사용
```
const birthday = dayjs('2000-01-01', 'YYYY-MM-DD').locale('zh-cn');
```
