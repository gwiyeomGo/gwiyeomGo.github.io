### 배경
코드가 3년 동안 많이 늘어났다
기존에는 빨리 만들기 위해 코드를 신경쓰지 않았는데
코드를 다시 보고 공통 함수등 활용해 개선이 필요하다고 느꼈다

### 실제 경험
> 사례1
알림톡이 어떤 상태의 회원에가 가는지 모르겠다
부분부분 개발하면서 어떤 알림톡은 미전환 회원에게 가고 어떤 알림톡은 활성 회원에게 전달된다.
개선 방법
회원정보를 부른다
회원정보가 미전환,활성 회원인지 확인 코드 포함된 조회 함수 생성
해당 함수를 회원 정보가 필요한 부분에 사용
