webpack 을 알기전에..

Webpack is a free and open-source module bundler for JavaScript. module bundler?

1.모듈(module) 자바스크립트 파일을 기능 단위로 모듈화하고 이것을 하나로 관리할 방법이 필요해졌다.

  1. 번들러(bundler) 의존성이 있는 모듈들을 하나(또는 여러 개)의 파일로 만들어주는 도구
